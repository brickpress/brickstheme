== BRICKS THEME FOR WORDPRESS ==

Contributors: Raphael Villanea support@brickpress.us Requires at least: 3.1 Tested up to: 3.4.1 Version: 1.0.0

== Description ==

Bricks is a clean, intuitive, flexible and responsive theme for WordPress. Inspired by WordPress, the Open Source Initiative and LEGO.

Theme Options Sections:

  • General Settings
  • Header Settings
  • Navigation Settings
  • Content Settings
  • Sidebars Settings
  • Footer Settings
  • Social Media Settings

Custom Widgets:

  • Bricks Tabs Widget
  • Bricks Search Widget
  • Bricks Category Post Widget

Theme Supports:

  • Custom Header
  • Custom Background
  • Post Formats: aside, audio, chat, gallery, image, link, quote, status, video
